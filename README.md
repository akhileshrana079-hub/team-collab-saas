# 🚀 Team Collaboration SaaS

A full-stack Team Collaboration SaaS built with React, Node.js, Express, and MongoDB.

## 🌐 Live Demo

Frontend: https://team-collab-saas-olive.vercel.app

Backend API: https://team-collab-saas.onrender.com

Swagger Docs:
https://team-collab-saas.onrender.com/api-docs

---

# Features

## Authentication

- Register
- Login
- JWT Authentication
- Protected Routes

## Workspace Management

- Create Workspace
- View Workspaces
- Delete Workspace

## Member Management

- Add Members
- Update Member Roles
- Remove Members

## Project Management

- Create Project
- View Projects
- Archive Project
- Delete Project

## Task Management

- Create Task
- Update Task Status
- Delete Task
- Priority Levels

## Dashboard

- Workspace Count
- Project Count
- Task Count
- Member Count

# Installation

## Clone Repository

```bash
git clone https://github.com/akhileshrana079-hub/team-collab-saas.git
```

## Backend

```bash
cd backend
npm install
npm run dev
```

## Frontend

```bash
cd client
npm install
npm run dev
```
